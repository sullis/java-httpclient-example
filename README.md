# java-httpclient-example


# System requirements

This project requires:
- Java 11.x
- Maven 3.6.1

# Java 11 development kit

- [Amazon Corretto 11.x](https://docs.aws.amazon.com/corretto/latest/corretto-11-ug/downloads-list.html)

# Build and run

```

./build.sh
./run.sh

```
